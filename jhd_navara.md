## 🎨 JHD Design Package System

| แพ็กเกจ | จำนวนแบบ | แก้ไข | เหมาะสำหรับ |
|---|---|---|---|
| 🥉 จัดให้ | 1 แบบ | ❌ ไม่แก้ | งบจำกัด งานเร่ง ลูกค้าไว้ใจทีม |
| 🥈 จัดไป | 1 แบบ | ✅ 2 ครั้ง | ต้องการปรับรายละเอียด มีภาพในใจ |
| 🥇 จัดเต็ม | 3 แบบ | ✅ 2 ครั้ง | ต้องการเปรียบเทียบ งาน Branding |
| 🏆 จัดหนัก | ตาม Scope | ตาม Scope | Premium / Luxury / Facade เต็มรูปแบบ |

**เงื่อนไขทุกแพ็กเกจ:**
- การแก้ไขเกินสิทธิ์ = มีค่าใช้จ่ายเพิ่ม
- เปลี่ยน Concept ใหม่ทั้งหมด = ประเมินใหม่
- ค่าออกแบบ = ค่าดำเนินการเพื่อผลิตกับ JHD เท่านั้น
- ไม่รวม: Source File / Editable File / AI File (มีค่าเพิ่ม)
- ค่าออกแบบหักเป็นส่วนลดได้เมื่อผลิตครบทุกรายการกับ JHD

## 🖥️ Graphic Design Workflow (ทีม)

```
Brief → เลือกแพ็กเกจ → Research & Reference
→ Design Direction → Artwork Development
→ Visual Presentation (Mockup/Perspective)
→ Feedback & Revision → Final Approval
→ Production File Preparation → ส่ง BIGM
```

**เครื่องมือที่ใช้:** Canva · AI Visualization · Graphic Software · Hybrid Workflow

**มาตรฐานงานทุกชิ้น:**
✔ อ่านง่าย เข้าใจได้ทันที | ✔ ใช้งานจริงได้ | ✔ เหมาะกับการผลิต | ✔ สมดุลด้านการออกแบบ

## 📐 Design Scale System™ (ค่าออกแบบ)

| Scale | ช่วงงาน | ค่าบริการ | ตัวอย่าง |
|---|---|---|---|
| 🟢 S | งานเล็ก Template แก้เล็กน้อย | 150–500 บาท | แก้ข้อความ ปรับสี |
| 🟡 M | Layout ใหม่ มี Mockup | 500–1,500 บาท | งานป้ายทั่วไป |
| 🟠 L | Branding หลายรายการ Visualization | 1,500–5,000 บาท | งานหน้าร้าน |
| 🔴 XL | Premium Concept Facade Creative Team | 5,000+ บาท | งาน DoFacade |

*(อ้างอิงมูลค่าโครงการ: S <3k | M 3k–15k | L 15k–30k | XL 30k+)*

## 📐 EXTENSION: Design Limitation — Hybrid Letter
