# PRODUCT KNOWLEDGE BASE: JHD 3D Standard Letter™

## 1. Overview
- **นิยาม:** ระบบตัวอักษรโลหะมาตรฐาน (18 SKU) ตั้งแต่ซิงค์ (เข้าถึงง่าย) ถึง สแตนเลส PVD (Luxury)
- **ระบบไฟ:** รองรับ Solid, Halo (ไฟออกหลัง), และ Frontlit (ไฟออกหน้า)

## 2. วัสดุหลัก (Main Materials)
- **ซิงค์ (Zinc):** น้ำหนักเบา รับสีพ่นดี เหมาะ SME
- **สแตนเลส (Stainless Steel):** ทนทาน เหมาะงานถาวร (ผิว: 2B, Hairline, งาช้าง, พ่นสี)
- **PVD Coating (Luxury):** สแตนเลสเคลือบสูญญากาศ (สี: ทอง, ดำ, โรสโกลด์)
- **อะคริลิค (เสริม):** ใช้หนุนหลัง, ปิดหลัง, หรือเป็นหน้าตัวอักษร

## 3. ระบบไฟ (Light System)
- **Solid:** ทึบ
- **Halo:** LED Module เรืองหลัง (ตัวอักษรมีมิติ)
- **Frontlit:** LED Module ส่องหน้า (เด่นชัด สว่าง)

## 4. ตารางสรุปกลุ่มสินค้า (18 SKU)
| กลุ่ม | ประเภท | จุดเด่น | การใช้งาน |
| :--- | :--- | :--- | :--- |
| **A (ไม่มีไฟ)** | ซิงค์/สแตนเลส/PVD | ราคาประหยัด - Luxury | ป้ายสำนักงาน, งานถาวร |
| **B (Halo)** | ซิงค์/PVD/Hairline | แสงเรืองรอง มีมิติ | Corporate, Luxury Brand |
| **C (Frontlit)** | ซิงค์/Hairline/PVD | สว่างเด่นชัด | Retail, Hotel, แบรนด์ที่ต้องการความโดดเด่น |

## 5. ลำดับราคา (Price Ranking: ต่ำ -> สูง)
1. ซิงค์ดิบ (Raw Zinc)
2. ซิงค์ทำสี / Halo โบ๋
3. สแตนเลสผิวธรรมชาติ / Hairline
4. สแตนเลสทำสี / Frontlit
5. สแตนเลส PVD (ทุกแบบ)

## 6. USP (จุดแข็ง)
- 18 SKU ครอบคลุมทุก Segment ตั้งแต่ SME ถึง Luxury
- PVD Coating สีทอง/ดำ/โรสโกลด์ ให้ผิวสัมผัสระดับสูงสุดที่วัสดุอื่นทำไม่ได้
- ระบบ LED Module คุณภาพสูง (ต่างจากรุ่น Hybrid ที่ใช้ LED Strip)
