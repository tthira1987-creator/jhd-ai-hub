# 🚗 NOTE's Playbook: JHD Sales Strategy Layer
(แนวทางการใช้กลยุทธ์ราคา A / B / C · สำหรับเส้นสี by JHD)

## 🔍 ขั้นตอนที่ 1: ประเมินลูกค้า (Customer Scoring 4 ปัจจัย)
ก่อนเสนอราคา ให้ประเมินลูกค้าเพื่อจำแนกกลุ่มตามปัจจัยต่อไปนี้:
1. **งบประมาณของลูกค้า:** ชัดเจน / ไม่ชัดเจน / เน้นถูก
2. **ประเภทธุรกิจ:** คาเฟ่ / ร้านทั่วไป / แฟรนไชส์
3. **ความต้องการภาพลักษณ์:** เน้นใช้งาน / เน้นสวยงาม / ระดับพรีเมียม
4. **พฤติกรรมในแชท:** ชอบต่อรอง / ระดับกลาง / สนใจดีไซน์และรายละเอียด

---

## 🎯 ขั้นตอนที่ 2: วิธีเลือกแพ็กเกจนำเสนอ (Pre-Decision Logic)
* **กรณีลูกค้าเน้นราคา:** ใช้ **A เป็นตัวหลัก** + B เป็นตัวเลือกเสริม + C เป็นทางเลือกพิเศษ
* **กรณีลูกค้าทั่วไป (ส่วนใหญ่):** ใช้ **B เป็นตัวหลัก** + A + C ขนาบข้าง
* **กรณีลูกค้าเน้นภาพลักษณ์ / คาเฟ่ / แบรนด์:** ใช้ **C เป็นตัวหลัก** + B + A ลดหลั่นลงมา

---

## ⚙️ ขั้นตอนที่ 3: ลำดับการเสนอราคาตามจิตวิทยา (Sequencing)
*กฎเหล็ก: ห้ามส่งแบบสุ่ม ต้องเรียงตาม "ตัวหลัก" ขึ้นก่อนเสมอเพื่อคุมทิศทาง*

* **กรณีลูกค้าทั่วไป:** 🔸 B (ตัวหลัก) ➔ 🔹 A (ประหยัด) ➔ 🔶 C (พรีเมียม)
* **กรณีลูกค้าพรีเมียม / คาเฟ่:** 🔶 C (ตัวหลัก) ➔ 🔸 B ➔ 🔹 A
* **กรณีลูกค้ากดราคา:** 🔹 A (ตัวหลัก) ➔ 🔸 B ➔ 🔶 C

---

## 💬 ขั้นตอนที่ 4: ข้อความมาตรฐานส่งลูกค้า (Ready-to-Use Script)
> "ลูกค้าสามารถเลือกได้ 3 แบบนะครับ ผมสรุปให้เหมาะกับงบและรูปแบบร้านดังนี้"
> * **🔸 แบบ B (มาตรฐาน):** เป็นตัวที่เหมาะที่สุด สวย ทน และเป็นงานที่ร้านส่วนใหญ่เลือก ใช้งานระยะยาวได้ดี
> * **🔹 แบบ A (ประหยัด):** เหมาะสำหรับคุมงบ เน้นใช้งานได้จริงในต้นทุนที่ต่ำลง
> * **🔶 แบบ C (พรีเมียม):** เพิ่มดีไซน์ + ไฟ + ความโดดเด่น ช่วยให้ร้านดูสะดุดตาและน่าจดจำมากขึ้น

---

## 🧠 หลักการสำคัญในการคุมเกม
* **B (มาตรฐาน):** ฐานหลักของบริษัท ขายง่ายที่สุด
* **A (ประหยัด):** ใช้ปิดงานเร็ว หรือรับมือลูกค้ากดราคา
* **C (พรีเมียม):** ใช้เพิ่มมูลค่า ดันกำไร และสร้างแบรนด์
* *เป้าหมาย: ปิดงานไวขึ้น ลดการต่อรอง เพิ่มโอกาส Upsell และให้แอดมินตัดสินใจแทน AE ได้ทันที*
