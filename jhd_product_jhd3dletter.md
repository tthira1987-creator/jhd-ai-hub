# PRODUCT KNOWLEDGE BASE: JHD 3D Letter Series™

## 1. ภาพรวม (Overview)
JHD 3D Letter Series™ คือกลุ่มสินค้าตัวอักษรนูน 3 มิติของ เส้นสี JHD ที่ครอบคลุมทุก Segment ตั้งแต่ระดับเข้าถึงได้จนถึงระดับ Luxury โดยมีโครงสร้างสินค้า 2 Series หลัก

## 2. โครงสร้างสินค้า (Product Architecture)
### Series 01 — JHD Hybrid Letter™
* **วัสดุ:** โครงสร้างหลักพลาสวูด (Plaswood) ตัด CNC + วัสดุหน้าหลากหลาย
* **จุดเด่น:** นวัตกรรมเฉพาะของ JHD ปรับแต่ง Option ได้ทีละ Layer
* **ความเหมาะสม:** SME, ร้านค้าที่ต้องการป้ายสวยในงบจัดการได้

### Series 02 — JHD 3D Standard Letter™
* **วัสดุ:** โลหะแท้ (ซิงค์ / สแตนเลส)
* **จุดเด่น:** มาตรฐานสูง ผิวสำเร็จพรีเมียม (PVD ทอง/ดำ/โรสโกลด์)
* **ความเหมาะสม:** Corporate, Luxury Brand, Office

## 3. Price Spectrum (ช่วงระดับราคา)
*ระดับจากเข้าถึงง่าย ไปจนถึง พรีเมียม:*
1. **Hybrid Base:** พลาสวูดดิบ (ประหยัดสุด)
2. **Hybrid + Face Material:** พลาสวูด + วัสดุหน้าพรีเมียม
3. **Standard Entry:** ซิงค์ / พ่นสี
4. **Standard Mid:** สแตนเลส 2B / Hairline
5. **Standard Top:** สแตนเลส PVD (ทอง / ดำ / โรสโกลด์) - พรีเมียมสุด

## 4. ตารางเปรียบเทียบ
| หัวข้อ | JHD Hybrid Letter™ | JHD 3D Standard Letter™ |
| :--- | :--- | :--- |
| **วัสดุหลัก** | พลาสวูด CNC + วัสดุหน้า | โลหะแท้ (ซิงค์/สแตนเลส) |
| **ระดับราคา** | เข้าถึงได้ ปรับตาม Layer | กลาง–สูง ตามวัสดุและผิว |
| **ระบบไฟ** | LED Strip (ไฟเส้น) | LED Module |
| **ดีไซน์** | ไม่จำกัด (UV Print + Wrap) | จำกัดตามผิววัสดุโลหะ |
| **การติดตั้ง** | DIY ได้ (Pre-drilled) | ช่างติดตั้งหลัก |
