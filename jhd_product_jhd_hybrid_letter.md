# PRODUCT KNOWLEDGE BASE: JHD Hybrid Letter™

## 1. Overview
- **Tagline:** ป้าย 3D ระดับ Mall ในราคาที่ทุกธุรกิจเข้าถึงได้
- **ลักษณะ:** อักษรนูน 3D โครงสร้างพลาสวูด CNC + วัสดุหน้าพรีเมียม
- **จุดเด่น:** ระบบ Layer ที่เลือก Upgrade ได้ตามงบ (เริ่มต้นจาก Layer 1)

## 2. Layer System (โครงสร้างการสั่งซื้อ)
### Layer 1: Base (โครงสร้าง)
- **วัสดุ:** พลาสวูด CNC ความหนา 6, 8, 10, 15, 20 mm
- **Finish:** (1) ดิบ (Raw) หรือ (2) เก็บขอบ+ทำสีข้าง (Finished)

### Layer 2: Face Option (ผิวหน้า)
- **เรียงจากงบประหยัดไปพรีเมียม:**
  1. พ่นสี (Flexibility สูง)
  2. อลูคอม/ACP (Outdoor)
  3. ซิงค์อะลู/ZincAlu (Premium Outdoor 10+ ปี)
  4. อะคริลิค (Premium Indoor)
  5. ลามิเนต (Interior)
  6. UV Print + 3M Wrap (Unlimited Graphic)

### Layer 3: Add-on (ตัวเสริม)
- **Sign Base Panel:** สำหรับติดตั้งและซ่อนสายไฟ
- **LED Strip Halo:** ไฟเรืองหลัง (คิดราคา: เส้นรอบรูป x 2)
- **Standoff + Pre-drilled:** ชุดติดตั้งแบบ DIY (Pre-drilled พร้อมติดตั้งเอง)

## 3. ระบบราคา (Pricing Formula)
- **Base Price:** [ขนาดความสูง] × [จำนวนตัวอักษร] × [อัตราตาม Layer วัสดุ]
- **LED Price:** [เส้นรอบรูป (cm)] × 2

## 4. USP (จุดแข็ง)
- DIY-Ready: มาร์คและเจาะรูจากโรงงาน ติดตั้งเองได้เหมือน IKEA
- Standoff Production: ผลิตเอง ต้นทุนต่ำ แข็งแรงสูง
- Design Freedom: รองรับ UV Print / Wrap ไม่จำกัดลวดลาย
